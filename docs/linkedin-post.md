# LinkedIn Post: FalkorDB on Snowflake Marketplace

Excited to share: **FalkorDB is now live on Snowflake Marketplace!** 🎉

After months of integration work, I've brought FalkorDB's graph database capabilities directly into Snowflake as a Native App. Users can now run Cypher queries on their Snowflake tables without moving data outside their security boundary.

Built using Snowflake Container Services with reference binding for secure data access. The integration handles everything from compute pool management to CSV staging for graph loading.
